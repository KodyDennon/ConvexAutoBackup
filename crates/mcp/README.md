# convex-autobackup-mcp

`convex-autobackup-mcp` is the Model Context Protocol helper package for ConvexAutoBackup automation. It is intended for AI-assisted and tool-driven workflows that need a narrow interface into backup, restore, and operational status surfaces.

Most users do not need this crate directly. Install the main `convex-autobackup` binary unless you are wiring ConvexAutoBackup into an MCP-capable tool environment.

## Install

```bash
cargo install convex-autobackup-mcp --locked
```
